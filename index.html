The Reverse of Differentiation

Quick Recap: Derivatives

Essential Derivative Rules

$\frac{d}{dx}[x^n] = nx^{n-1}$

$\frac{d}{dx}[c] = 0$ (where $c$ is constant)

$\frac{d}{dx}[cf(x)] = c \cdot f'(x)$

$\frac{d}{dx}[f(x) + g(x)] = f'(x) + g'(x)$

üß† Quick Quiz: Test Your Derivative Skills

Question 1: Find $\frac{dy}{dx}$ if $y = 3x^4 - 2x^2 + 5x - 1$

Answer: $\frac{dy}{dx} = 12x^3 - 4x + 5$

Working: Apply power rule to each term

  • $\frac{d}{dx}[3x^4] = 3 \cdot 4x^3 = 12x^3$
  • $\frac{d}{dx}[-2x^2] = -2 \cdot 2x = -4x$
  • $\frac{d}{dx}[5x] = 5$
  • $\frac{d}{dx}[-1] = 0$

Question 2: If $f(x) = x^3 - 6x^2 + 9x$, find the critical points.

Answer: Critical points at $x = 1$ and $x = 3$

Working:

  • $f'(x) = 3x^2 - 12x + 9$
  • Set $f'(x) = 0$: $3x^2 - 12x + 9 = 0$
  • $3(x^2 - 4x + 3) = 0$
  • $3(x - 1)(x - 3) = 0$
  • Therefore $x = 1$ or $x = 3$

Question 3: Find the equation of the tangent line to $y = x^2 + 2x$ at $x = 1$.

Answer: $y = 4x - 1$

Working:

  • $\frac{dy}{dx} = 2x + 2$
  • At $x = 1$: slope $= 2(1) + 2 = 4$
  • Point on curve: $(1, 1^2 + 2(1)) = (1, 3)$
  • Tangent: $y - 3 = 4(x - 1)$
  • Therefore: $y = 4x - 1$

Part 1: Basic Integration Concepts

Integration is the reverse process of differentiation!

If $\frac{dy}{dx} = f(x)$, then $y = \int f(x) \, dx$

1. Indefinite Integrals of $x^n$

Power Rule for Integration:

$$\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad \text{(where } n \neq -1\text{)}$$

Special Cases:

$\int 1 \, dx = x + C$

$\int x \, dx = \frac{x^2}{2} + C$

$\int \frac{1}{x} \, dx = \ln|x| + C$

üü¢ Easy Example

Find $\int x^3 \, dx$

Solution:

$\int x^3 \, dx = \frac{x^{3+1}}{3+1} + C = \frac{x^4}{4} + C$

Check: $\frac{d}{dx}\left[\frac{x^4}{4}\right] = x^3$ ‚úì

üü¢ Easy Example

Find $\int 5x^2 \, dx$

Solution:

$\int 5x^2 \, dx = 5 \int x^2 \, dx = 5 \cdot \frac{x^3}{3} + C = \frac{5x^3}{3} + C$

üü° Medium Example

Find $\int x^{-2} \, dx$

Solution:

$\int x^{-2} \, dx = \frac{x^{-2+1}}{-2+1} + C = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C$

2. Integrating Polynomials

General Rule:

$$\int (ax^n + bx^m + cx^p + d) \, dx = a\frac{x^{n+1}}{n+1} + b\frac{x^{m+1}}{m+1} + c\frac{x^{p+1}}{p+1} + dx + C$$

üü° Medium Example

Find $\int (3x^2 - 4x + 7) \, dx$

Solution:

$\int (3x^2 - 4x + 7) \, dx$

$= 3 \int x^2 \, dx - 4 \int x \, dx + 7 \int 1 \, dx$

$= 3 \cdot \frac{x^3}{3} - 4 \cdot \frac{x^2}{2} + 7x + C$

$= x^3 - 2x^2 + 7x + C$

üî¥ Hard Example

Find $\int \left(2x^3 - \frac{3}{x^2} + \sqrt{x} - 5\right) \, dx$

Solution:

Rewrite: $\int (2x^3 - 3x^{-2} + x^{1/2} - 5) \, dx$

$= 2 \cdot \frac{x^4}{4} - 3 \cdot \frac{x^{-1}}{-1} + \frac{x^{3/2}}{3/2} - 5x + C$

$= \frac{x^4}{2} + \frac{3}{x} + \frac{2x^{3/2}}{3} - 5x + C$

3. Finding f(x) Given f'(x) and a Point

Step-by-Step Method:

  1. Integrate $f'(x)$ to get $f(x) = \int f'(x) \, dx + C$
  2. Use the given point to find the constant $C$
  3. Write the final answer for $f(x)$

üü° Medium Example

Given $f'(x) = 6x^2 - 4x + 1$ and $f(1) = 5$, find $f(x)$.

Solution:

Step 1: Integrate $f'(x)$

$f(x) = \int (6x^2 - 4x + 1) \, dx$

$f(x) = 2x^3 - 2x^2 + x + C$

Step 2: Use $f(1) = 5$

$5 = 2(1)^3 - 2(1)^2 + 1 + C$

$5 = 2 - 2 + 1 + C$

$5 = 1 + C$

$C = 4$

Step 3: Final answer

$f(x) = 2x^3 - 2x^2 + x + 4$

üî¥ Hard Example

Given $\frac{dy}{dx} = 3x^2 - \frac{2}{x^3}$ and the curve passes through $(2, 10)$, find $y$.

Solution:

Step 1: Rewrite and integrate

$\frac{dy}{dx} = 3x^2 - 2x^{-3}$

$y = \int (3x^2 - 2x^{-3}) \, dx$

$y = x^3 - 2 \cdot \frac{x^{-2}}{-2} + C$

$y = x^3 + \frac{1}{x^2} + C$

Step 2: Use point $(2, 10)$

$10 = 2^3 + \frac{1}{2^2} + C$

$10 = 8 + \frac{1}{4} + C$

$C = 10 - 8.25 = 1.75$

Step 3: Final answer

$y = x^3 + \frac{1}{x^2} + \frac{7}{4}$

Part 2: Advanced Integration Concepts

1. Definite Integrals

Fundamental Theorem of Calculus:

$$\int_a^b f(x) \, dx = [F(x)]_a^b = F(b) - F(a)$$

where $F(x)$ is the antiderivative of $f(x)$

üü¢ Easy Example

Evaluate $\int_1^3 2x \, dx$

Solution:

$\int_1^3 2x \, dx = [x^2]_1^3$

$= 3^2 - 1^2$

$= 9 - 1 = 8$

üü° Medium Example

Evaluate $\int_0^2 (x^2 + 3x - 1) \, dx$

Solution:

$\int_0^2 (x^2 + 3x - 1) \, dx$

$= \left[\frac{x^3}{3} + \frac{3x^2}{2} - x\right]_0^2$

$= \left(\frac{8}{3} + 6 - 2\right) - (0)$

$= \frac{8}{3} + 4 = \frac{20}{3}$

üî¥ Hard Example

Evaluate $\int_1^4 \left(\sqrt{x} - \frac{1}{x^2}\right) \, dx$

Solution:

$\int_1^4 (x^{1/2} - x^{-2}) \, dx$

$= \left[\frac{x^{3/2}}{3/2} - \frac{x^{-1}}{-1}\right]_1^4$

$= \left[\frac{2x^{3/2}}{3} + \frac{1}{x}\right]_1^4$

$= \left(\frac{16}{3} + \frac{1}{4}\right) - \left(\frac{2}{3} + 1\right)$

$= \frac{14}{3} - \frac{3}{4} = \frac{56-9}{12} = \frac{47}{12}$

2. Area Under a Curve and x-axis

Area Formula:

$$\text{Area} = \int_a^b |f(x)| \, dx$$

Important: If $f(x) < 0$ in some intervals, split the integral!

Example: Area above x-axis

Find the area between $y = x^2$ and the x-axis from $x = 0$ to $x = 2$.

Solution:

Since $x^2 \geq 0$ for all $x$:

$\text{Area} = \int_0^2 x^2 \, dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3}$

Example: Area below x-axis

Find the area between $y = x^2 - 4$ and the x-axis from $x = -2$ to $x = 2$.

Solution:

Since $x^2 - 4 \leq 0$ for $-2 \leq x \leq 2$:

$\text{Area} = \int_{-2}^2 |x^2 - 4| \, dx = \int_{-2}^2 (4 - x^2) \, dx$

$= \left[4x - \frac{x^3}{3}\right]_{-2}^2$

$= \left(8 - \frac{8}{3}\right) - \left(-8 + \frac{8}{3}\right) = \frac{32}{3}$

üî¥ Complex Example: Mixed Areas

Find the total area between $y = x^3 - x$ and the x-axis from $x = -1$ to $x = 1$.

Solution:

Step 1: Find where $f(x) = 0$

$x^3 - x = 0 \Rightarrow x(x^2 - 1) = 0 \Rightarrow x = 0, \pm 1$

Step 2: Determine sign in each interval

  • For $-1 < x < 0$: $f(-0.5) = -0.125 + 0.5 = 0.375 > 0$
  • For $0 < x < 1$: $f(0.5) = 0.125 - 0.5 = -0.375 < 0$

Step 3: Split the integral

$\text{Area} = \int_{-1}^0 (x^3 - x) \, dx + \int_0^1 |x^3 - x| \, dx$

$= \int_{-1}^0 (x^3 - x) \, dx + \int_0^1 (x - x^3) \, dx$

$= \left[\frac{x^4}{4} - \frac{x^2}{2}\right]_{-1}^0 + \left[\frac{x^2}{2} - \frac{x^4}{4}\right]_0^1$

$= (0) - \left(\frac{1}{4} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{4}\right) - (0)$

$= \frac{1}{4} + \frac{1}{4} = \frac{1}{2}$

3. Areas Between Curves and Lines

Area between curve and line:

$$\text{Area} = \int_a^b |f(x) - g(x)| \, dx$$

where $f(x)$ is the curve and $g(x)$ is the line

üü° Example: Curve above line

Find the area between $y = x^2 + 1$ and $y = 2x + 1$ from $x = 0$ to $x = 2$.

Solution:

Step 1: Find intersection points

$x^2 + 1 = 2x + 1$

$x^2 = 2x$

$x^2 - 2x = 0$

$x(x - 2) = 0$

So $x = 0$ or $x = 2$

Step 2: Determine which function is above

At $x = 1$: $y_1 = 1 + 1 = 2$, $y_2 = 2 + 1 = 3$

So line is above curve

Step 3: Calculate area

$\text{Area} = \int_0^2 [(2x + 1) - (x^2 + 1)] \, dx$

$= \int_0^2 (2x - x^2) \, dx$

$= \left[x^2 - \frac{x^3}{3}\right]_0^2$

$= 4 - \frac{8}{3} = \frac{4}{3}$

üî¥ Complex Example

Find the area enclosed by $y = x^2 - 2x$ and $y = x$.

Solution:

Step 1: Find intersection points

$x^2 - 2x = x$

$x^2 - 3x = 0$

$x(x - 3) = 0$

So $x = 0$ or $x = 3$

Step 2: Determine which is above

At $x = 1$: curve gives $1 - 2 = -1$, line gives $1$

So line is above curve

Step 3: Calculate area

$\text{Area} = \int_0^3 [x - (x^2 - 2x)] \, dx$

$= \int_0^3 (3x - x^2) \, dx$

$= \left[\frac{3x^2}{2} - \frac{x^3}{3}\right]_0^3$

$= \frac{27}{2} - 9 = \frac{9}{2}$

4. Areas Between Two Curves

Area between two curves:

$$\text{Area} = \int_a^b |f(x) - g(x)| \, dx$$

where $a$ and $b$ are intersection points

General Method:

  1. Find intersection points by solving $f(x) = g(x)$
  2. Determine which curve is above in each interval
  3. Set up integral(s) with upper curve minus lower curve
  4. Evaluate the definite integral(s)

üü° Medium Example

Find the area between $y = x^2$ and $y = 4 - x^2$.

Solution:

Step 1: Find intersections

$x^2 = 4 - x^2$

$2x^2 = 4$

$x^2 = 2$

$x = \pm\sqrt{2}$

Step 2: Determine which is above

At $x = 0$: $y_1 = 0$, $y_2 = 4$

So $y = 4 - x^2$ is above $y = x^2$

Step 3: Calculate area

$\text{Area} = \int_{-\sqrt{2}}^{\sqrt{2}} [(4 - x^2) - x^2] \, dx$

$= \int_{-\sqrt{2}}^{\sqrt{2}} (4 - 2x^2) \, dx$

$= \left[4x - \frac{2x^3}{3}\right]_{-\sqrt{2}}^{\sqrt{2}}$

$= \left(4\sqrt{2} - \frac{4\sqrt{2}}{3}\right) - \left(-4\sqrt{2} + \frac{4\sqrt{2}}{3}\right)$

$= 8\sqrt{2} - \frac{8\sqrt{2}}{3} = \frac{16\sqrt{2}}{3}$

üî¥ Hard Example

Find the area enclosed by $y = x^3 - 3x$ and $y = x$.

Solution:

Step 1: Find intersections

$x^3 - 3x = x$

$x^3 - 4x = 0$

$x(x^2 - 4) = 0$

$x(x - 2)(x + 2) = 0$

So $x = -2, 0, 2$

Step 2: Determine which is above in each interval

For $-2 < x < 0$: At $x = -1$, cubic gives $-1 + 3 = 2$, line gives $-1$

For $0 < x < 2$: At $x = 1$, cubic gives $1 - 3 = -2$, line gives $1$

Step 3: Split into two integrals

$\text{Area} = \int_{-2}^0 [(x^3 - 3x) - x] \, dx + \int_0^2 [x - (x^3 - 3x)] \, dx$

$= \int_{-2}^0 (x^3 - 4x) \, dx + \int_0^2 (4x - x^3) \, dx$

$= \left[\frac{x^4}{4} - 2x^2\right]_{-2}^0 + \left[2x^2 - \frac{x^4}{4}\right]_0^2$

$= (0) - (4 - 8) + (8 - 4) - (0)$

$= 4 + 4 = 8$

5. The Trapezium Rule

When to use: When the integral cannot be evaluated exactly or when working with data points.

Trapezium Rule Formula:

$$\int_a^b f(x) \, dx \approx \frac{h}{2}[y_0 + 2(y_1 + y_2 + \cdots + y_{n-1}) + y_n]$$

where $h = \frac{b-a}{n}$ and $y_i = f(a + ih)$

Visual Understanding

The trapezium rule approximates the area under a curve by dividing it into trapezoids.

Each trapezoid has:

  • Width: $h = \frac{b-a}{n}$
  • Parallel sides: $y_i$ and $y_{i+1}$
  • Area: $\frac{h}{2}(y_i + y_{i+1})$

üü° Example: Using 4 strips

Use the trapezium rule with 4 strips to approximate $\int_0^2 x^2 \, dx$.

Solution:

Step 1: Find $h$ and $x$-values

$h = \frac{2-0}{4} = 0.5$

$x$-values: $0, 0.5, 1, 1.5, 2$

Step 2: Calculate $y$-values

$y_0 = 0^2 = 0$

$y_1 = 0.5^2 = 0.25$

$y_2 = 1^2 = 1$

$y_3 = 1.5^2 = 2.25$

$y_4 = 2^2 = 4$

Step 3: Apply formula

$\int_0^2 x^2 \, dx \approx \frac{0.5}{2}[0 + 2(0.25 + 1 + 2.25) + 4]$

$= 0.25[0 + 2(3.5) + 4]$

$= 0.25[11] = 2.75$

Exact value: $\int_0^2 x^2 \, dx = \frac{8}{3} = 2.667...$

Error: $|2.75 - 2.667| = 0.083$

üî¥ Example: Data points

The table shows values of a function $f(x)$. Use the trapezium rule to estimate $\int_0^3 f(x) \, dx$.

x 0 0.5 1 1.5 2 2.5 3
f(x) 1 1.2 1.8 2.1 2.9 3.4 4.2

Solution:

$h = 0.5$ (given by the $x$-values)

$\int_0^3 f(x) \, dx \approx \frac{0.5}{2}[1 + 2(1.2 + 1.8 + 2.1 + 2.9 + 3.4) + 4.2]$

$= 0.25[1 + 2(11.4) + 4.2]$

$= 0.25[1 + 22.8 + 4.2]$

$= 0.25 \times 28 = 7$

üí° Key Points about Trapezium Rule:

  • Accuracy: More strips ‚Üí better approximation
  • Error: Tends to overestimate for concave up curves, underestimate for concave down
  • Alternative: Simpson's rule (more accurate but requires even number of strips)
  • Applications: Engineering, physics, economics when exact integration is impossible

Real-World Applications of Integration

üöó Physics: Distance and Velocity

Problem: A car's velocity is given by $v(t) = 3t^2 + 2t$ m/s. Find the distance traveled from $t = 0$ to $t = 4$ seconds.

Solution:

Distance = $\int_0^4 v(t) \, dt = \int_0^4 (3t^2 + 2t) \, dt$

$= \left[t^3 + t^2\right]_0^4$

$= 64 + 16 = 80$ meters

üí∞ Economics: Consumer Surplus

Problem: Demand function: $p = 100 - 2q$, equilibrium at $q = 20$. Find consumer surplus.

Solution:

Equilibrium price: $p = 100 - 2(20) = 60$

Consumer Surplus = $\int_0^{20} (100 - 2q) \, dq - 20 \times 60$

$= \left[100q - q^2\right]_0^{20} - 1200$

$= 2000 - 400 - 1200 = 400$

üèóÔ∏è Engineering: Volume of Revolution

Problem: Find the volume when $y = \sqrt{x}$ from $x = 0$ to $x = 4$ is rotated about the x-axis.

Solution:

$V = \pi \int_0^4 (\sqrt{x})^2 \, dx = \pi \int_0^4 x \, dx$

$= \pi \left[\frac{x^2}{2}\right]_0^4 = \pi \cdot 8 = 8\pi$ cubic units

üå°Ô∏è Environmental Science: Pollution Accumulation

Problem: Pollution rate: $P(t) = 50e^{0.1t}$ tons/year. Find total pollution over 10 years.

Solution:

Total = $\int_0^{10} 50e^{0.1t} \, dt$

$= 50 \left[\frac{e^{0.1t}}{0.1}\right]_0^{10}$

$= 500(e - 1) \approx 859$ tons

üéØ Exam Strategy & Common Mistakes:

‚úÖ Do:

  • Always add constant of integration for indefinite integrals
  • Check your integration by differentiating
  • Sketch graphs to visualize area problems
  • Find intersection points carefully
  • Split integrals when function changes sign
  • Show all working steps clearly
  • Use correct notation: $\int$ not $\sum$

‚ùå Don't:

  • Forget the constant $C$ in indefinite integrals
  • Ignore negative areas (use absolute value)
  • Mix up upper and lower limits
  • Forget to find intersection points
  • Use trapezium rule when exact answer is possible
  • Rush through area calculations
  • Forget units in real-world problems

üß† Final Challenge Problem

Problem: The region bounded by $y = x^2$, $y = 2x$, and $x = 3$ is rotated about the y-axis. Find the volume of the solid formed.

Solution:

Step 1: Find intersection of $y = x^2$ and $y = 2x$

$x^2 = 2x \Rightarrow x^2 - 2x = 0 \Rightarrow x(x-2) = 0$

So $x = 0$ or $x = 2$

Step 2: Set up the volume integral using shells method

For $0 \leq x \leq 2$: height = $2x - x^2$

For $2 \leq x \leq 3$: height = $x^2 - 2x$ (but this gives negative height)

Actually, we need to be more careful about the bounded region...

Alternative approach using washers:

Convert to functions of $y$: $x = \sqrt{y}$ and $x = \frac{y}{2}$

$V = \pi \int_0^4 \left[(\sqrt{y})^2 - \left(\frac{y}{2}\right)^2\right] dy$

$= \pi \int_0^4 \left(y - \frac{y^2}{4}\right) dy$

$= \pi \left[\frac{y^2}{2} - \frac{y^3}{12}\right]_0^4$

$= \pi \left(8 - \frac{64}{12}\right) = \pi \left(8 - \frac{16}{3}\right) = \frac{8\pi}{3}$
